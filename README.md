# Textarea-Character-Counter

A simple web application that implements a real-time character counter for a textarea input using HTML, CSS, and JavaScript. The counter updates dynamically as the user types, displaying the number of characters entered out of the maximum allowed limit. Once the limit of 200 characters is reached, further input is restricted and a warning message is displayed.

Tools Used
HTML5
CSS3
JavaScript

Features:
Real-time character counting
Displays typed characters and remaining limit (e.g., 150 / 200 characters)
Prevents input beyond the maximum character limit
Displays a warning message when the limit is reached
